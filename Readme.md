# HBESD Agent-Based Model

This repository contains the Python-based Agent-Based Model (ABM) simulation accompanying the paper: **From Bayesian Surprise to Structural Phase Transition: The Mechanics of Adaptive Criticality**. 

This simulation models the thermodynamic pressure of Bayesian Surprise ($S_i$) and Elastic Capacity ($C_i$) across a predictive processing network, demonstrating structural phase transitions and cascading failures.

## Requirements
* Python 3.x
* Libraries used: os, random, math, multiprocessing, concurrent, numpy, network, scipy, pandas, matplotlib

## How to Run
1. Clone this repository.
2. Install the required dependencies.
3. Execute the model by running:
   ```bash
   python ABM_5_2_final.py
